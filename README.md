#  Прогнозирование виновности автомобиля в ДТП

##  Описание проекта

Данный проект посвящён разработке модели машинного обучения для прогнозирования вероятности того, что автомобиль окажется виновником дорожно-транспортного происшествия.

Проект реализован как прототип системы оценки рисков для сервиса каршеринга.  
Цель — предсказывать риск ДТП перед началом поездки и использовать эту информацию для предупреждения водителей или принятия бизнес-решений.

---

##  Бизнес-задача

Каршеринговая компания хочет:

- Оценивать риск ДТП перед подтверждением поездки
- Предупреждать водителей о повышенном риске
- Потенциально учитывать риск при расчёте тарифа

Модель должна предсказывать вероятность того, что автомобиль окажется виновником аварии.

---

##  Используемые данные

В проекте используются несколько связанных таблиц:

- **case_ids** — идентификаторы ДТП и год происшествия
- **vehicles** — характеристики автомобиля (возраст, тип коробки передач и др.)
- **collisions** — информация об условиях ДТП (погода, освещение, тип дороги)
- **parties** — сведения об участниках и признак виновности (целевая переменная)

### Ограничения выборки:
- Только участники с `party_type = 'car'`
- Исключены случаи с минимальными повреждениями
- Используются данные начиная с 2012 года
- Возраст автомобиля обязателен для анализа

- Python

- NumPy

---

##  Этапы работы

1. Подключение к базе данных и извлечение данных
2. Предобработка и очистка данных
3. Исследовательский анализ данных (EDA)
4. Анализ корреляций
5. Подготовка признаков
6. Обучение моделей машинного обучения
7. Оценка качества модели
8. Интерпретация результатов

---

##  Целевая переменная

Бинарная классификация:

- `1` — автомобиль признан виновником ДТП  
- `0` — автомобиль не признан виновником  

---

##  Результат

В рамках проекта построена модель классификации, позволяющая оценить вероятность виновности автомобиля в ДТП на основе исторических данных.

Модель может быть использована как часть системы риск-менеджмента каршерингового сервиса.
